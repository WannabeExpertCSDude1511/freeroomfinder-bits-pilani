# Free Room Finder - BITS Pilani

Free Room Finder is a web application that helps students at **BITS Pilani** locate available classrooms during free hours. It is designed for students looking for a convenient place to study, work, or rest between classes.

## Features

- Search for free classrooms by:
  - Day
  - Starting hour
  - Required availability (1–4 hours)
  - Preferred building(s)
- Supports multiple building selection.
- Color-coded room badges by building.
- Fast, responsive interface built with Next.js and TypeScript.

## Tech Stack

- **Next.js**
- **React**
- **TypeScript (.tsx)**
- **Tailwind CSS**
- **DaisyUI**

The timetable data is extracted from the official PDF timetable using a TypeScript (`.ts`) script and stored as a JSON file for efficient querying.
